# 🔘 Button Set Redesign

## 📌 Overview

This project was completed as **Task 1** during my **UI/UX Design Internship at SAM AI Technologies**.

The objective was to design a scalable and reusable button component library following modern UI design principles, ensuring consistency and accessibility across digital products.

---

## 🎯 Objective

- Design reusable button components
- Create multiple button variants and states
- Maintain visual consistency
- Build a component-based design system

---

## ✨ Features

- Primary Buttons
- Secondary Buttons
- Outline Buttons
- Disabled State
- Hover State
- Consistent Border Radius
- Responsive Component Design
